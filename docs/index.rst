Mutalyzer
=========

Mutalyzer is a tool primarily designed to check descriptions of sequence
variants according to the Human Genome Sequence Variation Society
(`HGVS <https://www.hgvs.org/>`_) `guidelines <https://varnomen.hgvs.org/>`_.
